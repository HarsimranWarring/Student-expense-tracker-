# Student Expense Tracker 💰

A professional desktop application for tracking student expenses with a modern GUI, budget management, and analytics.

## Features ✨

- **Expense Tracking**: Add, edit, delete, and view expenses
- **Category Management**: Organize expenses by categories with color coding
- **Budget Alerts**: Set monthly budgets and get alerts when spending exceeds limits
- **Analytics Dashboard**: View spending insights with charts and statistics
- **Data Export**: Export expenses to CSV format
- **Data Backup**: Automatic backup and restore functionality
- **Dark Theme UI**: Modern, eye-friendly interface
- **Persistent Storage**: SQLite database for reliable data persistence
